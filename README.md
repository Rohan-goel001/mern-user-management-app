# MERN User Management App

## 🚀 Features

* Add user
* View users
* Update user
* Delete user

## 🛠 Tech Stack

* Frontend: React
* Backend: Node.js, Express
* Database: MongoDB

## 📂 Project Structure

mern-project/
├── backend/
├── frontend/

## ▶️ How to Run

### Start MongoDB

mongod

### Start Backend

cd backend
node server.js

### Start Frontend

cd frontend
npm start

## 📌 Project Description

This is a full-stack MERN (MongoDB, Express, React, Node.js) application that performs CRUD (Create, Read, Update, Delete) operations on user data.

The application allows users to:

* Add new users
* View all users
* Update existing user details
* Delete users

It demonstrates the integration of a React frontend with a Node.js and Express backend, connected to a MongoDB database. The project focuses on building a complete end-to-end workflow with REST APIs and dynamic data handling.

This project was developed as part of learning full-stack development using the MERN stack.
